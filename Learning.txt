Using The Debug Module :

1) Run on command prompt for debug logs using the namespace like 
    set DEBUG=rest-api-nodejs-mongodb:server & node ./bin/www
        or
    set DEBUG=rest-api-nodejs-mongodb:* & node ./bin/www
        or
    set DEBUG=* & node ./bin/www
    where namespace is declared as :
    var debug = require('debug')('rest-api-nodejs-mongodb:server');

2) For each new model we create these files
    eg. 1) models/ProductModel.js
        2) controllers/ProductController.js
        3) routes/product.js
